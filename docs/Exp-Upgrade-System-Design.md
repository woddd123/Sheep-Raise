# 经验值与升级系统规划

## 当前状态
- 已有 `level`、`exp` 状态
- 经验条显示在控制面板
- 每次升级需要 100 EXP

---

## 获取经验值的方式

### 1. 日常运营类（持续性）

| 行为 | 经验值 | 说明 |
|------|--------|------|
| 羊进食（吃草/吃料） | +1-2 | 每完成一次进食循环 |
| 羊成功产毛（剪毛） | +5 | 每次剪羊毛 |
| 羊成功繁育 | +10 | 每次新羊出生 |
| 清理粪便 | +2 | 每次清理 |

### 2. 成就里程碑类（一次性）

| 成就 | 经验值 | 说明 |
|------|--------|------|
| 首次购买羊 | +3 | 第一只羊 |
| 首次剪毛 | +5 | 第一次收羊毛 |
| 羊群规模达标（5/10/20只） | +10/20/30 | 达成规模里程碑 |
| 累计出售羊毛 x50 | +15 | 卖掉第一批羊毛 |
| 首次繁育成功 | +8 | 第一只小羊诞生 |

### 3. 突发事件类（偶发性）

| 事件 | 经验值 | 说明 |
|------|--------|------|
| 羊生病后康复 | +8 | 经历生病并存活 |
| 暴风雨存活 | +5 | 天气事件无损失 |
| 发现走失羊（自动回栏） | +3 | 羊自行回归 |

## 升级奖励与用途

### 等级解锁内容

| 等级 | 解锁内容 | 说明 |
|------|----------|------|
| Lv.2 | 羊圈上限+4（8只）| 从4只扩展到8只 |
| Lv.2 | 饲料槽功能 | 可建造饲料槽喂羊 |
| Lv.3 | 羊圈上限+4（12只）| 继续扩展容量 |
| Lv.3 | 繁育功能 | 成年羊可繁育 |
| Lv.4 | 羊圈上限+4（16只）| 容量持续扩展 |
| Lv.4 | 高级牧草 | 牧草恢复更快 |
| Lv.5 | 羊圈上限+4（20只）| 大型牧场规模 |
| Lv.5 | 羊毛品质提升 | 剪毛获双倍 |
| Lv.6+ | 重复Lv.5奖励 | 每级+4容量 |

### 每级固定奖励
- 金币奖励：Lv.1→Lv.2 +50金币，Lv.2→Lv.3 +100金币（递增）
- 羊毛储存上限提升（防止溢出）
- 自动出售功能解锁（Lv.5+）

---

## 平衡性建议

1. **经验曲线**：后期升级需要更多行为积累，避免玩家过早满级
2. **日常 vs 成就**：日常行为经验少但持续，成就要有挑战性但奖励丰厚
3. **分流设计**：让玩家可以选择"专注繁育"或"专注产毛"等不同路线获得经验
